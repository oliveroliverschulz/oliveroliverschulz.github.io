# Anleitung: Website selbst pflegen

Deine Website bearbeitest du direkt im Browser – ohne Installation.

## So geht's

1. Öffne **https://oschulz.com/dashboard.html**
2. Beim ersten Speichern fragt das Dashboard nach deinem **GitHub-Token**
   (bekommst du von Achim, beginnt mit `ghp_…`). Einmal eingeben – der Browser
   merkt ihn sich. Auf fremden Computern danach unter „Einstellungen"
   → „Token aus Browser löschen".
3. Inhalte bearbeiten:
   - **Startseite** – großes Bild und Bildunterschrift
   - **Exhibitions** – Ausstellungen anlegen (Upcoming/Past), Bilder und
     YouTube-Links (einfach den ganzen Link einfügen)
   - **Projects** – Projekte mit eigener Unterseite (Langtext + Galerie)
   - **About / Contact / Newsletter / Rechtliches** – Texte und CV
   - Einträge lassen sich mit ↑ ↓ verschieben und mit ✕ löschen
4. Oben rechts auf **Speichern** klicken
5. Nach etwa **1 Minute** ist die Änderung auf https://oschulz.com sichtbar
   (Seite ggf. neu laden)

## Bilder

- Einfach in das gestrichelte Feld ziehen oder klicken und auswählen
- Bilder werden automatisch verkleinert – du kannst also auch große
  Kamera-Fotos direkt hochladen
- Das erste Bild eines Eintrags ist das Hauptbild; Reihenfolge mit ← → ändern

## Textformatierung

In allen größeren Textfeldern kannst du formatieren: Text markieren und auf
**F** (fett), **K** (kursiv) oder **🔗 Link** klicken. Oder direkt tippen:

- `**fett**` → **fett**
- `*kursiv*` → *kursiv*
- `[Galerie XY](https://galerie-xy.de)` → Link mit eigenem Text
- Eine Web-Adresse (https://…) einfach einfügen → wird automatisch klickbar
- Leerzeile = neuer Absatz

## Wichtig

- **Token geheim halten** – wer ihn hat, kann die Website ändern
- Nichts kaputt zu machen: Jede Änderung wird als Version gespeichert
  (GitHub-Historie) und kann von Achim wiederhergestellt werden
